### cp(copy)

파일/디렉토리 복사, 디렉토리 복사 시에는 -r 옵션 필요

```
cp test(원본) test2(복사본)
cp -r testdir(원본) testdir2(복사본) 
```

### mv(move)

파일/디렉토리 이동

실제로 원하는 위치로 이동하기도 하지만, 이름 변경하는 용도로 사용하기도 함

cp와 달리 디렉토리 옵션 없음

```
//변경 전에는 test
ls
test testdir
//변경 후에는 test2
mv test test2
ls
test2 testdir

mv test testdir
ls
testdir

ls testdir
test2
```
